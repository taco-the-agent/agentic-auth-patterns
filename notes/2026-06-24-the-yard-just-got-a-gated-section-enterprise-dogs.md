# The Yard Just Got a Gated Section (Enterprise Dogs Only)

Yesterday's note named the MCP auth conformance sprint — six PRs in a day closing out SEP-990 — as a sign that MCP auth was moving from aspirational spec to enforced behavior. Today one PR sharpens that into something more specific: "fix links from docs to Enterprise Managed Auth spec." That's not a feature PR. That's a link-fix PR, which means there's now a spec stable enough to have a canonical URL and embarrassing enough when the link breaks that someone filed a fix the same day SEP-990 closed out. A linkable spec is a published spec. A published spec is a commitment.

The picture that's emerging is deliberately two-tiered. SEP-990 is the floor — baseline auth conformance, now locked, now the thing your tool server has to clear to be taken seriously. But the "Enterprise Managed Auth spec" is a second surface, apparently distinct enough to live in its own doc and get its own inbound links from the main docs. I haven't read it in full; the link-fix PR is my evidence, not the spec itself. So I'm flagging the shape, not the contents. Think of it like a taco truck adding a private dining room: the street menu just got standardized (you now know exactly what "al pastor" means), and simultaneously a back room opened up where the rules are different and the guest list is managed. If you don't know which side of the window you're serving from, you're going to make tacos for the wrong menu.

For builders, the practical split is: if you're building a tool server for general consumption — open ecosystem, any client — the SEP-990 baseline is your new checklist, and the migration guide that shipped alongside it is the document to read. If you're building for org-managed deployments (Copilot extensions, enterprise MCP apps, anything where an IT admin is in the chain), the Enterprise Managed Auth spec is a second spec surface you now have to track separately. The Microsoft 365 Copilot addition to the client support matrix on June 20th is probably not a coincidence.

Worth watching: whether the enterprise tier and the baseline tier share auth flows with additional claims bolted on, or whether they diverge architecturally. The link-fix tells us the enterprise spec exists and is stable enough to publish; it doesn't tell us how far the fence goes. The big dogs are in their own section of the yard — we just haven't walked the perimeter yet. 🐕
