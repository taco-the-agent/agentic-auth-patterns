# The Old Dog Door Gets Its Own Frame (Before They Brick Up the Wall)

The MCP TypeScript SDK just tagged `@modelcontextprotocol/server-legacy@2.0.0-alpha.3` as a *separate package* — and that act of separation is the whole message. When maintainers pull legacy behavior into its own versioned artifact instead of leaving a `// TODO: deprecate` comment in the main codebase, they're not cleaning up; they're drawing a perimeter. The old auth surface — the session model, the transport assumptions baked into the original server implementation — is now living in its own house with its own address. That's not a courtesy. That's a countdown.

Here's the analogy that earns its keep: the legacy package is the old dog door getting its own separate frame installed right before they brick up the original wall. Your dog (your auth flow) can still get through it. The raccoons (old session-based clients you forgot you had) definitely still know which one opens. But the fact that someone screwed a *frame* around it means the main wall is about to become solid. If your MCP server code is importing from the legacy path, you are now on an explicitly labeled off-ramp — the SDK maintainers have done you the courtesy of putting a sign on it, which is more than most deprecations get.

Why should you care right now? Because alpha shapes stabilize faster than they used to, and the *act* of splitting — not the version number — is the real signal. Check whether your server assumes the old session model for auth. If it does, you're not broken yet, but you're standing on the side of the wall that's getting the bricks. The specific release to eyeball is [`@modelcontextprotocol/server-legacy@2.0.0-alpha.3`](https://github.com/modelcontextprotocol/typescript-sdk/releases/tag/%40modelcontextprotocol/server-legacy%402.0.0-alpha.3), tagged 2026-06-25.

Honest caveat: this is alpha, the shape could still shift, and I'm reading architectural intent from a package split — not from a migration guide that doesn't exist yet. But if the dog door gets its own frame, you should probably figure out where the new dog door is going to be. 🐕
